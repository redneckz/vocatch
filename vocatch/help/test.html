8. Проверка работоспособности ПО

Информация по установке ПО находится в разеделе "Установка ПО". Перед проверкой желательно ознакомиться с другими разделами данной документации.

Шаги базовой проверки:

  1. Произвести установку;
  2. Подключить и проверить источник видеосигнала;
  3. Установить HASP ключ в USB-порт системы, на которой происходит проверка;
  4. Создать и активировать режим со следующими параметрами (см. раздел "Описание интерфейса ПО"): "Размер буфера" = 1, "Просмотр видеоряда" = да, "Непрерывный" = нет, "Сохранение измерений" = нет;
  5. Активировать "Демонстрационный VCL-модуль" и "Стандартный CALC-модуль";
  6. Для "Демонстрационного VCL-модуля" задать следующие параметры: "Папка слайдов" = ./demo-slides, "Размер слайда" = 512;
  7. Для "Стандартного CALC-модуля" задать следующие параметры: "Радиус фильтрации" = 8, "Жесткость порога" = 75, "Уровень отладки фильтра" = 0;
  8. Запустить цикл работы из главного окна;
  9. По истечении некоторого времени в главном окне должно появиться изображение прямоугольного объекта с поправками, линия поправки должна проходить рядом с осью симметрии объекта. В окне лога должны появиться два сообщения о начале и конце цикла работы;
  10. Создать и активировать режим со следующими параметрами: "Размер буфера" = 1, "Просмотр видеоряда" = да, "Непрерывный" = да, "Сохранение измерений" = нет;
  11. Активировать "Pentacon VCL-модуль";
  12. Для "Pentacon VCL-модуля" задать следующие параметры: "Размер слайда" = 512, "Формат видеосигнала" = 2;
  13. Запустить цикл работы из главного окна;
  14. В главном окне должно происходить непрерывное отображение поступающего от источника видеосигнала с поправками. В случае возникновения ошибок в расчёте поправок следует отрегулировать параметры "Стандартного CALC-модуля" или активировать и настроить "Расширенный CALC-модуль".

В случае возникновения ошибок сохранять копии файлов лога (".\bin\log\*.txt") в отдельную директорию для последующего изучения и внесения изменений в ПО.

Шаги полной проверки:

  1. Выполнить шаги базовой проверки;
  2. Подключить к локальной сети управляющую машину;
  3. Создать и активировать режим со следующими параметрами: "Размер буфера" = 1, "Просмотр видеоряда" = да, "Непрерывный" = да, "Сохранение измерений" = да, "Рабочий каталог" = !не забудьте указать каталог!;
  4. Открыть главное окно и окно лога;
  5. Послать с управляющей машины команду старта (см. раздел "Особенности ПО");
  6. В главном окне должно происходить непрерывное отображение поступающего от источника видеосигнала с поправками. В окне лога должны появиться соответствующие записи.
  7. В выбранном рабочем каталоге должен появиться подкаталог с текущей датой, внутри которого файл "variant0.epop" с поправками. Размер файла должен постоянно увеличиваться до прихода с управляющей машины команды остановки.
